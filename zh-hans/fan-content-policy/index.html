1. 适用范围

本《粉丝内容政策》（以下简称“本政策”）规范基于 Sushi Ben 以及 Big Brane Studios, Inc.（以下简称“Big Brane”“我们”）所拥有的相关知识产权所创作和分享的非官方粉丝内容。

我们重视社区的创造力，并希望在明确且合理的指引下支持粉丝表达。本政策说明哪些类型的粉丝内容是允许的，以及适用哪些限制。

2. 定义

2.1 “Sushi Ben IP”是指 Big Brane 就 Sushi Ben 所拥有或控制的全部知识产权，包括但不限于角色、名称、地点、故事元素、美术作品、音乐、音频、视频、标识以及其他专有内容。

2.2 “粉丝内容”是指由您创作的、基于 Sushi Ben IP 或包含 Sushi Ben IP 的原创非官方内容，包括但不限于美术作品、视频、直播、游戏片段以及其他类似创意作品。

3. 许可授予

3.1 在您遵守本政策的前提下，Big Brane 授予您一项有限的、非独占的、可撤销的、不可转让的许可，以创作和分享基于 Sushi Ben IP 的粉丝内容。

3.2 本政策不转让 Sushi Ben IP 的任何所有权。

3.3 Big Brane 可随时以任何理由并自行决定撤销该许可。

4. 允许的使用

在遵守本政策限制的前提下，您可以：

4.1 创作并分享非商业性的粉丝内容。

4.2 在 Twitch、YouTube 或类似服务等平台上直播或以其他方式公开展示 Sushi Ben。

4.3 将游戏视频、片段、录制内容或类似内容上传至 YouTube、TikTok、Twitch 或类似服务等平台。

4.4 仅通过平台支持的功能将视频和直播变现，例如：
- 广告收入；
- 合作伙伴计划收入；
- 订阅；以及
- 打赏。

4.5 仅以能够明确表明您的粉丝内容是非官方且并非由 Big Brane 拥有、赞助或认可的方式使用 Sushi Ben IP。

5. 限制

您不得：

5.1 销售、许可或以其他方式商业化利用粉丝内容，包括实体或数字商品，无论规模大小。

5.2 对使用 Sushi Ben IP 的粉丝内容进行大规模生产、商业分发或销售。

5.3 使用 Kickstarter 或 Indiegogo 等众筹平台来创作、筹资或销售基于 Sushi Ben IP 的商品或其他产品。

5.4 将粉丝内容表述为由 Big Brane 官方发布、授权、批准、赞助或认可。

5.5 将 Sushi Ben IP 用于与违法、诽谤、仇恨、歧视、骚扰或以其他方式对个人或群体有害的内容相关联的情形。

5.6 以可能合理损害 Sushi Ben 或 Big Brane 声誉的方式使用 Sushi Ben IP，包括不当、冒犯性或误导性的描绘。

5.7 将 Sushi Ben IP 用于与宗教或政治信息或图像相关联的情形。

5.8 将 Sushi Ben IP 用于与 NFT、加密货币、基于区块链的资产或类似技术相关联的情形。

6. 必要声明

粉丝内容必须以清晰且合理的方式包含以下声明：

“这是非官方粉丝内容，与 Big Brane Studios 无关联，也未获得 Big Brane Studios 的认可或支持。”
